# **What is a EC2 Instance?**
#### EC2 stands for Elastic Compute Cloud which is a web service provided by Amazon AWS. An EC2 Instance is a on-demand service by AWS in which you can use a instance based on your memmory and CPU requirment needed and then you will pay for how long you use that service for.

#### Log in to your AWS account or create an account if you don't have one.
